# 内建工具介绍

此包定义的内建工具（built-in tools）是指在 Agent 运行时中预定义的、可直接使用的工具。这些工具通常提供了常见的文件、命令操作功能，可以让 Agent 快速完成一些任务。

目前所需的内建工具包括：

## 文件操作工具
注意，文件类工具的操作权限均在工作目录之内，且不支持符号链接，以避免安全风险。如果需要操作工作目录之外的文件，需要用户二次确认
1. `list_files`：列出指定目录下的文件和子目录，支持多层级。
2. `read_file`：读取指定文件的内容，支持按行、指定行数读取。
3. `write_file`：将内容写入指定文件，支持按行、替换插入。
4. `search_file`：在指定目录及其子目录中搜索包含特定关键词的文件，支持正则表达式搜索。
5. `grep_file`：在指定文件中搜索包含特定关键词的行。
6. `delete_file`：删除指定文件，此功能需要二次确认。
7. `move_file`：移动或重命名文件。
8. `copy_file`：复制文件到指定位置。

## 命令执行工具
1. `exec_command`：在指定环境中执行命令，支持 shell、cmd 命令。
2. `check_command`：检查命令是否存在于系统中，返回命令路径或版本信息，例如检查是否存在python环境，可结合exec_command使用 python 能力。
3. `list_processes`：列出当前系统中正在运行的进程，支持过滤和排序。
4. `kill_process`：终止指定进程，此功能需要二次确认。
5. `get_system_info`：获取系统信息，如操作系统版本、CPU、内存等。
6. `http_request`：发送 HTTP 请求，支持 GET、POST 等方法，并返回响应内容。

## 其他工具
1. `web_search`：使用搜索引擎查询信息，返回搜索结果摘要，首批支持 tavily、serpapi、bing，预留API key配置。
